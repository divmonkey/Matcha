✅ Core Rules (rules.md)

  • Mobile-First Design - Responsive for all devices
  • Tailwind CSS Only - No custom CSS files
  • Precision Units - Use px units, not rem
  • Icon System - Heroicons Outline + Geometric SVGs from svgrepo.com
  • Human Faces - Filtered by gender via uifaces.co API
  • No Leaks - Frontend cannot touch Django models directly
  • Image Optimization - All images validated for size & alt tags
